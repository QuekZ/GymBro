This is a Website made for the WPV Webtechnologies. 

This was made with basic HTML, CSS and some Javascript. It enables the user to get a Training Plan according to their needs with 4 different types of plans. 
The Plan can be saved and printed too.

This Project was made by Mehmet Can Celik, David Heiner and Karl Dufner
